##  Возможности

-  Поддержка **VLESS**, **Hysteria2**, **Trojan**
-  Управление множеством серверов с быстрым переключением
-  Добавление серверов из буфера обмена (`vless://`, `hysteria2://`, `trojan://`) или из файла / URL
-  **Drag & Drop** – поддерживается перетаскивание ссылок прямо на окно или в трей‑зону
-  Проверка пинга для каждого сервера
-  **Сплит‑туннелирование** с гибкими списками маршрутов (домены / IP / CIDR)
-  **Kill Switch** – блокировка всего интернета при падении прокси
-  Авто‑переподключение и последовательное переключение на следующий сервер при сбое
-  Исключения по приложениям (путь `.exe` или имя процесса) и доменам (точное имя / суффикс)
-  **TUN‑режим** – весь системный трафик идёт через виртуальный интерфейс (максимальная анонимность)
-  Отображение скорости соединения в реальном времени (↓ / ↑)
-  **Горячие клавиши** – вкл/выкл прокси, маршрутизации, TUN‑режима, выхода, быстрый выбор списка маршрутизации
-  Автозапуск вместе с Windows (настраивается)
-  Сворачивание в системный трей с динамическим меню (серверы, списки маршрутов)
-  Интерфейс на **русском** и **английском** языках
-  **Экспорт / импорт** полной конфигурации (серверы, маршруты, настройки) в файл `.taa`
-  Уведомления в стиле «тостов» о подключении, отключении, ошибках
-  **QR‑код** – быстрое сканирование ссылки телефонами (из контекстного меню сервера)
-  **Автоматическая проверка обновлений** и установка прямо из приложения
-  Шифрование учётных данных через **Windows DPAPI** + ACL на файлы через `icacls`

##  Использование

### Основное окно

- **Левая панель** – список серверов (с типом, статусом, меткой «по умолчанию»), кнопки добавления из буфера, импорта конфигов, редактор списков маршрутизации и настройки.
- **Правая панель** – подробная информация о выбранном сервере: имя (можно изменить кликом), адрес с опцией скрытия IP, проверка пинга, кнопка «По умолчанию», удаление, большая кнопка подключения/отключения, а также блок управления сплит‑туннелированием.
- **Скорость соединения** – в верхней части отображается текущая скорость загрузки и отдачи.

### Добавление серверов

- **Из буфера обмена** – скопируйте ссылку (`vless://`, `hysteria2://`, `trojan://`) и нажмите «➕ Из буфера».
- **Импорт конфигов** – выберите файл `.txt` / `.conf` / `.json` с одной или несколькими ссылками (поддерживается Base64).
- **По URL** – скопируйте в буфер обмена прямую http‑ссылку на текстовый файл с ключами и нажмите «➕ Из буфера» – программа загрузит и распарсит его.
- **Drag & Drop** – перетащите ссылку или текстовый файл на главное окно либо на специальную зону в трее (📌).

### Контекстное меню сервера (правая кнопка мыши)

- **Подключиться / Отключиться**
- **Установить по умолчанию** (будет выбран при запуске)
- **QR‑код для телефона** – открывает окно с QR‑кодом ссылки (удобно для Shadowrocket, v2rayNG, Streisand)
- **Копировать ссылку** – прямо из окна QR‑кода
- **Удалить сервер**

### Управление списками маршрутизации

- Перейдите в «Сплит‑лист» левого меню.
- Создавайте, удаляйте, переименовывайте списки и редактируйте их содержимое.
- Каждая строка – домен (`youtube.com`), IP (`1.1.1.1`) или подсеть (`192.168.0.0/16`).
- При включённом сплит‑туннелировании через прокси идут **только** адреса из активного списка.
- Быстрое переключение между списками – через комбо‑бокс, горячие клавиши или трей‑меню.

### TUN‑режим

- Включается в настройках (потребуются права администратора – приложение само предложит перезапуститься).
- Создаёт виртуальный интерфейс `taa-tun0` и перенаправляет **весь** трафик системы через прокси.
- Обеспечивает максимальную защиту – ни одно приложение не сможет обойти прокси.
- Для отключения снимите флажок и перезапустите соединение.

### Горячие клавиши

- Настраиваются в разделе «Настройки → Горячие клавиши».
- Можно назначить до четырёх основных действий (Toggle, Routing, TUN, Exit) и до десяти привязок к файлам маршрутизации.
- Комбинации должны включать хотя бы одну модификаторную клавишу (Ctrl, Alt, Shift).
- При конфликте или невозможности регистрации программа предупредит.

### Экспорт / импорт конфигурации

- Настройки → вкладка «Конфиг».
- **Экспорт:** выберите, что сохранить (сервера и/или списки маршрутизации), задайте имя `.taa`‑файла.
- **Импорт:** загрузите ранее сохранённый файл – настройки, серверы и списки будут объединены (оконные координаты не перезаписываются).

### Обновления

- Настройки → вкладка «Версия».
- Кнопка «Проверить обновления» сравнивает локальную версию с последней на GitHub.
- При наличии обновления появится кнопка «Обновить» – загрузка и распаковка происходят автоматически.
- Для работы обновления на территории РФ рекомендуется предварительно подключиться к прокси.

##  Безопасность

- **DPAPI** – все чувствительные данные (серверы, настройки) шифруются и привязаны к текущей учётной записи Windows.
- **icacls** – временные и постоянные конфигурационные файлы получают права доступа только для текущего пользователя.
- **Случайный порт** – локальный прокси запускается на случайном порту из диапазона **10000–60000** при каждом запуске.
- **Kill Switch** – при падении sing‑box мгновенно блокируется весь сетевой трафик через правило «TaaProxy_KillSwitch» в брандмауэре Windows.
- **Проверка целостности** – конфигурационные файлы защищены от чтения и подмены.

##  Установка и запуск

1. Скачайте последний релиз со [страницы загрузок](https://github.com/bububebe0/TaaProxy/releases).
2. Распакуйте архив в любую папку (рекомендуется не в системные директории).
3. Запустите `TaaProxy.exe`.  
   - При первом запуске будут созданы папки `data\` (настройки, зашифрованная база серверов) и `list\` (списки маршрутизации).
   - Для использования **TUN‑режима** нужны права администратора – при попытке включения программа сама предложит перезапуститься от имени администратора.
4. Рекомендуется периодически проверять обновления во вкладке «Версия» и включить авто‑подключение к прокси для надёжного скачивания.
