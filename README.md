## ✨ Возможности

- 🔌 Поддержка **VLESS** и **Hysteria2** (через sing-box)
- 🗂 Управление несколькими серверами с быстрым переключением
- 📋 Добавление серверов из буфера обмена (vless:// / hy2://)
- 📡 Проверка пинга для каждого сервера
- 🌍 Отображение флага страны сервера
- 🔀 Сплит-туннелирование с настраиваемыми списками маршрутов
- 🛡 **Kill Switch** — блокировка интернета при падении прокси
- 🔄 Авто-переключение на следующий сервер при сбое
- 🚫 Исключения по приложениям и доменам
- 🔒 Шифрование учётных данных через **Windows DPAPI**
- 🧩 Случайный порт и авторизация на локальном прокси при каждом запуске
- 🔁 Автозапуск с Windows
- 🗃 Сворачивание в системный трей
- 🌐 Интерфейс на **русском** и **английском** языках
- 🐞 Режим отладки с rotating-логами

---
## ⚙️ Настройки

| Параметр | Описание |
|---|---|
| Сплит-туннелирование | Только указанные домены идут через прокси |
| Kill Switch | Блокирует весь интернет при падении прокси |
| Авто-переключение | Переключается на следующий сервер при сбое |
| DNS (Системный / DoH / DoT) | Выбор DNS-резолвера |
| Исключения приложений | Указанные процессы обходят прокси |
| Исключения доменов | Указанные домены обходят прокси |
| Автозапуск | Запуск при старте Windows (через реестр) |

## 🔒 Безопасность

- Учётные данные и конфигурация шифруются через **Windows DPAPI** — доступны только текущему пользователю
- `config.json` удаляется с диска сразу после передачи конфига в sing-box
- ACL на временных файлах ограничивает доступ другим учётным записям
- Случайный порт и учётные данные для локального прокси генерируются при каждом запуске
- Защита от запуска нескольких экземпляров через Mutex
