## 🖥️ Использование

### Основное окно

- **Левая панель** – список серверов, кнопки добавления, импорта, настроек и выхода.
- **Правая панель** – информация о выбранном сервере (имя, адрес с опцией скрытия IP), проверка пинга, установка по умолчанию, удаление, кнопка подключения.
- **Блок маршрутизации** – свитч сплит‑туннелирования и выбор активного списка.

### Добавление серверов

- **Из буфера обмена** – скопируйте ссылку (`vless://`, `hysteria2://`, `trojan://`,) и нажмите «➕ Из буфера».
- **Импорт конфигов** – выберите файл `.txt` / `.conf` / `.json` с одной или несколькими ссылками (поддерживается Base64).
- **По URL** – скопируйте http‑ссылку на текстовый файл с конфигами в буфер обмена и нажмите «➕ Из буфера» – программа загрузит и распарсит его.

### Управление списками маршрутизации

- Нажмите «Сплит‑лист» в левом меню.
- Вы можете создать новый список, удалить или переименовать существующий.
- Формат: каждая строка – домен (например `youtube.com`) или IP/CIDR (`1.1.1.1`, `192.168.0.0/16`).
- При включённом сплит‑туннелировании через прокси идут **только** домены/IP из активного списка.

### TUN‑режим

- Включается в настройках (требует прав администратора – приложение предложит перезапуститься).
- Создаёт виртуальный интерфейс `taa‑tun0` и маршрутизирует через него **весь** сетевой трафик системы.
- **Обеспечивает максимальную анонимность** – никакое приложение не может обойти прокси.
- Для отключения снимите галочку в настройках и перезапустите соединение.

### Горячие клавиши

Настройки → вкладка «Горячие клавиши».  
Поддерживаются комбинации с **Ctrl, Alt, Shift**. Одни и те же клавиши можно назначать на разные действия.

| Действие | Описание |
|----------|----------|
| Вкл/Выкл прокси | Подключение / отключение текущего сервера |
| Вкл/Выкл маршрутизацию | Переключение сплит‑туннелирования |
| Вкл/Выкл TUN‑режим | Активация / деактивация TUN‑режима (требует прав администратора) |
| Выход из программы | Полное завершение |

### Экспорт / импорт конфигурации

- Настройки → вкладка «Конфиг».
- **Экспорт:** выберите, что сохранить (сервера и/или списки маршрутизации), укажите имя `.taa`‑файла.
- **Импорт:** загрузите ранее сохранённый файл – настройки, сервера и списки будут объединены (оконные координаты не перезаписываются).

## 🔒 Безопасность

- **DPAPI** – все учётные данные (сервера, настройки) шифруются и привязаны к текущему пользователю Windows.
- **icacls** – временные конфигурационные файлы и файлы данных получают ACL, запрещающий доступ другим учётным записям.
- **Случайный порт** – локальный прокси‑сервер каждый раз запускается на новом порту (10000‑60000).
- **Kill Switch** – при падении sing‑box блокируется весь сетевой трафик через брандмауэр Windows (правило `TaaProxy_KillSwitch`).

2. Распакуйте архив в любую папку.
